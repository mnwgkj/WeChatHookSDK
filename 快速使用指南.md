# WeChatHookSDK 快速使用指南

## 🚀 一键构建和测试

### 环境要求
- **macOS** with Xcode 16+
- **越狱iPhone** (iOS 12.0+)
- **SSH工具** (可选: ldid)

### 快速开始

#### 1️⃣ 构建项目
```bash
# 克隆或下载项目后
cd WeChatHookSDK

# 一键构建 (支持Xcode 16)
./build.sh
```

#### 2️⃣ 部署测试
```bash
# 交互式部署和测试
./test.sh

# 或一键完整测试
./test.sh full
```

#### 3️⃣ 配置使用
打开微信 → 设置 → 红包助手，配置：
- ✅ **总开关**: 开启
- ⚡ **延迟抢包**: 0毫秒 (极速)
- ⚡ **查询间隔**: 0毫秒 (极速)
- ✅ **极速抢包**: 开启
- ❌ **详细日志**: 关闭 (性能优化)

## 📋 构建流程详解

### build.sh 功能
1. **环境检查** - Xcode版本、项目文件
2. **清理构建** - 删除旧文件，创建新目录
3. **编译项目** - Release配置，极速优化
4. **拷贝产物** - dylib和plist文件
5. **自动签名** - 使用ldid签名
6. **基本测试** - 验证文件完整性

### test.sh 功能
1. **检查文件** - 验证构建产物
2. **设备连接** - SSH连接测试
3. **部署文件** - 上传到正确位置
4. **功能测试** - 重启SpringBoard
5. **结果验证** - 检查加载状态

## 🛠️ 手动构建 (可选)

如果自动脚本有问题，可以手动执行：

```bash
# 1. 使用Xcode构建
xcodebuild -project HookSDK.xcodeproj \
           -target HookSDK \
           -configuration Release \
           -sdk iphoneos \
           ARCHS=arm64 \
           IPHONEOS_DEPLOYMENT_TARGET=12.0

# 2. 查找生成的dylib
find build -name "*.dylib"

# 3. 签名
ldid -S path/to/libHookSDK.dylib

# 4. 部署到设备
scp libHookSDK.dylib root@device_ip:/Library/MobileSubstrate/DynamicLibraries/
scp Doc/libHookSDK.plist root@device_ip:/Library/MobileSubstrate/DynamicLibraries/

# 5. 重启SpringBoard
ssh root@device_ip "killall SpringBoard"
```

## ⚡ 性能优化配置

### 极速模式设置
```
延迟抢包(毫秒): 0          # 理论极限速度
查询详情间隔(毫秒): 0      # 最快查询
极速抢包: 开启              # 使用缓存timingId
详细日志: 关闭              # 减少性能开销
自动修改资料: 关闭          # 避免额外操作
```

### 多账号配置
```
主号 (iPhone A):
- 自动抢包: 开启
- 通知小号: 开启
- 查询详情: 开启

小号 (iPhone B):
- 自动抢包: 关闭
- 接收通知: 开启
- 极速模式: 开启
```

## 🔧 故障排除
